New session
tmux new-session -s work

Show session
tmux ls 

Back to Tmux
tmux a

Kill session
tmux kill-session -t myname

Break Freeze
ctrl-q

Panes (splits)
%  vertical split
"  horizontal split
o  swap panes

<> left right ^ up ...
 
Toggle last active pane
;

Exit pane
^b d

Show time
^b t

New session
tmux new-session -s HTB

Show session
tmux ls 

Back to Tmux
tmux a

tmux rename-session [new-name]
Kill session
tmux kill-session -t myname

Break Freeze
ctrl-q

Panes (splits)
%  vertical split
"  horizontal split
o  swap panes

Enable mouse mode
set -g mouse on

<> left right ^ up ...
 
Toggle last active pane
;

Exit pane
^b d

Need skill of timing: 

Show time
^b t

Show pane numbers
^b q
